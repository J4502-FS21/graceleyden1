Tips for new coders


HTML-Specific Checklist

1. The homepage HTML file is titled: index.html

2. All HTML documents includes a CSS link

3. Use opening closing tags for elements in code

4. Attributes have proper quotes

5. Class names should not contain spaces


General Tips

1. Readability and usability: Is your website easy to navigate? Is the content relevant and easy to access and understand? These are questions you should ask yourself when designing a website. It's essentisl for any good website to be easy to naviagate because it makes the viewer want to stay on the website longer. It also makes the viewer more interested in the content and want to leave with more knowledge on the topic.

2. Try and try again: Coding can be frustrating sometimes. There may be times where you cannot figure out why a code won't validate. Usually, it's right in fron of your face. So, keep trying. The feeling of accomplishment after is rewarding.

3. Images: Make sure your images are resized to scale for the page.

4. Alternate language: It's important to add a short descrption of images for those who are disabled.

5. Stay organized: It's easy to lose your files if you're not organized. Start all projects with a new folder and add a index.html, CSS and other pages in one file.
